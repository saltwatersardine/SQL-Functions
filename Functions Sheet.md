```sql
RIGHT JOIN is a function that helps you combine information from two tables based on a common column but from the second table. It's like having two toy boxes again, but this time you take all the toys from the second box and any toys from the first box that match the toys from the second box. -->
```

```sql
OUTER JOIN is a function that helps you combine information from two tables, including any information that doesn't have a match in the other table. You take all the toys from both boxes and include any matching toys. If there are any toys that don't have a match in the other box, they'll still be included, but with NULL values.
```
